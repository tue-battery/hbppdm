# Heavy‑Ball Projected Primal‑Dual Method (HBPPDM)
This repository provides a lightweight implementation of the Heavy‑Ball Projected Primal‑Dual Method (HBPPDM) for constrained optimization. The method accelerates a standard projected primal–dual algorithm by adding heavy‑ball momentum, which improves convergence speed when solving constrained linear and quadratic problems.
The HBPPDM algorithm and its theoretical properties are based on peer‑reviewed research.
